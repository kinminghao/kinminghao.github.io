发表于 2017-05-18 | 分类于 Java

简述

HashMap 是我们在 java 中最频繁出现的一种数据存储工具. 它以 键-值 对的方式来存储数据,作为一个高效存储工具,它很频繁的受到了我们的使用.
但是其内部实现原理很多人都不是特别清楚,在这里我们一起研究一下.

属性

  • table:使用 Node<K,V> 对象进行数据的存储是一个链表,它有四个属性,分别是 哈希值, key, value, 指向下一个节点的指针.
  • entrySet: 键值对的集合来存储对应的集合数据.
  • size: 表示键值对集合的大小
  • modCount 修改次数,和 ArrayList 类似,表示修改次数,用以确保同步.
